# 🍔 Online Food Ordering and Delivery System 🚚

A modern web-based application that allows users to browse food items, place orders, and get them delivered efficiently. The system supports multiple roles including **User, Admin, Restaurant, and Delivery Partner**, ensuring a smooth and real-time food ordering experience.

---

## 🔗 Clone Repository

```bash
git clone https://github.com/Sharmakirti1404/online-food-ordering-and-delivery-system.git
```

---

## 🌟 Features

### 👤 User

* Browse food items 🍕
* Add items to cart 🛒
* Place orders 📦
* Online / Cash on Delivery 💳
* Track orders 📍

### 🛠️ Admin

* Manage orders
* View system data
* Control users

### 🍽️ Restaurant Panel

* View incoming orders
* Accept & prepare food
* Manage menu

### 🚚 Delivery Panel

* View assigned orders

## ⚙️ Installation & Setup

1. Clone the repository:

```bash
git clone https://github.com/Sharmakirti1404/online-food-ordering-and-delivery-system.git
```

2. Navigate to the project folder:

```bash
cd online-food-ordering-and-delivery-system
```

3. Install dependencies (if required):

```bash
pip install -r requirements.txt
```

4. Run the project:

```bash
python app.py
```

5. Open in browser:

```
http://127.0.0.1:5000
```
